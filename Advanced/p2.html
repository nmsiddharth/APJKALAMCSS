CSS CURSOR

CSS Cursor property is used to define cursor type when the mouse moves over a certain area or over a link in the webpage.

EXAMPLE

This is first paragraph.( AUTO)

This is second paragraph.(Default)

This is third paragraph.(NONE)

This is fourth paragraph.(CONTEXT-MENU)

Thi is fifth paragraph.(HELP)

This is sixth paragraph.(POINTER)

This is seventh paragraph. (PROGRESS)
